Launcher Documentation
---------------------------------------------------

`Launcher <http://launcher.apps.os.dcs.gla.ac.uk/>`_ is a tool that lets you create and run jobs on Glasgow's IDA Cluster.


This documentation is a work in progress. Feel free to `open an issue or make a pull request <http://docs.launcher.apps.os.dcs.gla.ac.uk/faq.html#how-can-update-correct-this-documentation>`_
to help help improve it!
